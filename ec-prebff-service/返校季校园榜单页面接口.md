# 返校季校园榜单页面接口

> OP编号：OP-37443  
> 后端服务：ec-prebff-service  
> 站点：仅 US 站  
> 平台：PC + H5

## 1. 接口信息

| 项目 | 说明 |
|------|------|
| URL | `GET /pages/campus/ranking` |
| 协议 | HTTPS |
| Content-Type | application/json |

### 1.1 请求 Headers

| Header | 必填 | 默认值 | 说明 |
|--------|------|--------|------|
| `token` | 是 | — | 用户登录凭证 |
| `device_id` | 否 | "" | 设备ID |
| `ym_id` | 否 | — | 匿名ID |
| `y_version` | 否 | — | 客户端版本号 |
| `y_platform` | 否 | pc | 平台：pc / h5 |
| `y_language` | 否 | en_US | 语言：en_US（英文）/ zh_CN（中文） |

> **多语言：** 接口根据 `y_language` 返回对应语言的文案。传 `zh_CN` 返回中文，其余返回英文。

## 2. 响应结构

### 2.1 顶层结构

| 字段 | 类型 | 说明 |
|------|------|------|
| `code` | Integer | 状态码，0 表示成功 |
| `message` | String | 状态信息 |
| `body` | Object | 业务数据 |

### 2.2 body 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `page_title` | String | 页面大标题 |
| `reward_text` | String | 奖励提示文案 |
| `rules_link_text` | String | 规则链接文案（点击跳转静态页面，链接前端自行配置） |
| `update_info` | String | 更新频率和截止说明 |
| `rankings` | Array\<RankingListVO\> | 校园排名榜单列表（固定 3 个） |
| `gashapon` | GashaponVO | 扭蛋机模块 |
| `bottom_cta` | BottomCtaVO | 底部引导模块 |
| `product_groups` | Array\<ProductGroupVO\> | 商品推荐分组列表 |

### 2.3 RankingListVO

| 字段 | 类型 | 说明 |
|------|------|------|
| `type` | String | 榜单类型：`popularity` / `activity` / `invitation` |
| `title` | String | 榜单标题（如 "Campus Rising Stars"） |
| `subtitle` | String | 榜单副标题（如 "Which campus is building the biggest Yami community?"） |
| `note` | String | 榜单注释（如 "*Ranked by the number of..."） |
| `schools` | Array\<SchoolRankVO\> | 学校排名列表，最多 8 个。无数据时为空数组 `[]` |

### 2.4 SchoolRankVO

| 字段 | 类型 | 说明 |
|------|------|------|
| `rank` | Integer | 排名（1~8） |
| `school_name` | String | 学校名称（根据语言返回中文/英文名） |

### 2.5 GashaponVO

| 字段 | 类型 | 说明 |
|------|------|------|
| `link` | String | 扭蛋机跳转相对路径（如 `fun/activity/gachamachine/633`） |
| `title` | String | 主标题（参与人数已格式化，如 "100+ people have already joined the capsule machine!"） |
| `subtitle` | String | 副标题 |
| `button_text` | String | 按钮文案 |
| `participant_count` | Long | 格式化后的参与人数（向下取整到百位，最低 100，不足时为 0） |

> **参与人数展示逻辑：**
> - `participant_count` 已格式化为百位整数（100, 200, 300...）
> - 若值为 0，表示参与人数不足 100，前端不展示人数相关文案
> - `title` 中已将 `{count}` 替换为实际格式化后的数字

### 2.6 BottomCtaVO

| 字段 | 类型 | 说明 |
|------|------|------|
| `title` | String | 引导标题（如 "Help Your School Climb the Rankings!"） |
| `button_text` | String | 按钮文案（如 "Start Referring Now"）。按钮行为：页面内锚点跳转 |

### 2.7 ProductGroupVO

| 字段 | 类型 | 说明 |
|------|------|------|
| `sortOrder` | Integer | 分组排序 |
| `title` | String | 分组标题（如 "Upperclassmen Recommend"） |
| `modules` | Array\<ProductModuleVO\> | 子模块列表 |

### 2.8 ProductModuleVO

| 字段 | 类型 | 说明 |
|------|------|------|
| `sortOrder` | Integer | 模块排序 |
| `title` | String | 模块标题 |
| `subTitle` | String | 模块副标题 |
| `products` | Array\<ItemVO\> | 商品列表（结构同 StoryPage 商品卡片） |

## 4. 边界情况说明

| 场景 | 返回表现 |
|------|---------|
| 榜单无数据（活动未开始） | `rankings` 返回 3 个完整结构，`schools` 为空数组 `[]` |
| 扭蛋机参与人数不足 100 | `participant_count` 为 0，`title` 中 {count} 替换为 0。前端判断为 0 时不展示人数文案 |
| 商品配置为空 | `product_groups` 为空数组 `[]` |
| 商品已下架/无库存 | 该商品不会出现在 `products` 列表中（后端自动过滤） |
| 文案配置为空 | `page_title`、`reward_text` 等文案字段返回空字符串 `""` |

## 5. 对接注意事项

1. **多语言**：通过 `y_language` header 控制，接口只返回一种语言的文案
2. **扭蛋机链接**：返回的是相对路径 `fun/activity/gachamachine/{lot_id}`，前端拼接域名
3. **"View Rules & Details" 按钮**：文案由接口返回（`rules_link_text`），跳转目标由前端配置静态页面链接
4. **"Start Referring Now" 按钮**：文案由接口返回（`bottom_cta.button_text`），行为是页面内锚点跳转（前端自行处理）
5. **商品卡片**：`products` 中的 ItemVO 结构与 StoryPage 商品卡片完全一致，可复用现有组件
6. **榜单固定 3 个**：`rankings` 数组始终返回 3 个元素，按 popularity → activity → invitation 顺序
